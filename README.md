This is my undergraduate research thesis for my B.S. in Computer Science at Georgia Southern University.

I have always been passionate about art, particularly painting. Thus, I wanted to tackle an issue in the realm of computational aesthetics.

This project is meant to serve as a proof-of-concept for a tool that can allow for art history research to be done at scale, specifically within individual movements.
While the classification task can be done by humans, art assessing requires an extremely high level of education, and as a result, there are very few individuals qualified to do so. This doesn't even account for the fact that they would need to assess thousands and thousands of works to make this possible.

Feature extraction methods such as edge detection will be utilized to assess the typical structure of objects within impressionist paintings, such as people, buildings, houses, etc.
A convolutional neural network is then trained on these proccessed images with the endgoal os being able to differentiate between impressionist and non-impressionist paintings.
